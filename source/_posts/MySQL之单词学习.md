| 单词                 | 意思                           |
| -------------------- | ------------------------------ |
| alter                | 改变                           |
| table                | 表                             |
| alter table          | 更改表格                       |
| primary              | 主要的、重要的                 |
| key                  | 钥匙                           |
| primary key          | 主键                           |
| delate               | 删除                           |
| delete from          | 删除                           |
| drop                 | 滴；下降                       |
| drop table           | 删除数据表、删除基本表、表删除 |
| drop primary key     | 删除主键                       |
| drop index           | 删除索引                       |
| create               | 创造、创建                     |
| create table         | 创建表                         |
| insert               | 插入                           |
| insert into          | 插入数据                       |
| values               | 价值观                         |
| auto_increment       | 自动递增                       |
| modify               | 修改、调整                     |
| unique               | 唯一的                         |
| database             | 数据库                         |
| use                  | 使用                           |
| show                 | 显示                           |
| select               | 选择                           |
| show database        | 显示数据库                     |
| show create database | 显示创建数据库                 |
| show create          | 显示创建                       |
| show create table    | 显示创建表                     |
| set                  | 设置                           |
| character set        | 字符集                         |
| update               | 更新                           |
| order by             | 排序依据                       |
| order                | 命令; 顺序                     |
| between ...and...    | 两者之间                       |
|                      |                                |
|                      |                                |
|                      |                                |
